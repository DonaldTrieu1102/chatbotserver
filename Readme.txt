Các bước chạy project:
- Bước 1: Mở project bằng visual studio code
- Bước 2: excute file sql trong thư mục sql trong database có tên manage-store
- Bước 3: sửa username và password trong file utils/database
- Bước 4: nhập lệnh: npm install
- Bước 5: nhập lệnh: node app.js

Trường hợp lỗi không run được: nhập: npm2 start
